LambdaKraft

This is an implementation of the analysis named Polyvariant Flow Analysis with Higher-ranked Polymorphic Types and Higher-order Effect Operators created by Jurriaan Hage and Stefan Holdermans.

Grammar

Expression ::= Variable | Abstraction | Application | Value | IF-Statement

Type ::= BoolType | Arrow

Term Name Term Syntax (Paper) Term Representation (Textually)
Variable ::=  $$ x^{i} $$ Var i
Abstraction ::=  $$ \mathtt{\lambda}\ \mathrm{Variable:Type}\ .\ \mathrm{Expression} $$ Abs (Var i Type) Expression
IF-Statement ::=  $$ \mathtt{if}\ \mathrm{Expression}\ \mathtt{then}\ \mathrm{Expression}\ \mathtt{else}\ \mathrm{Expression} $$If Expression Expression Expression
Value ::=  $$ \mathtt{True}\ |\ \mathtt{False} $$ VTrue | VFalse
Application ::=  $$\mathrm{Expression}\ \mathrm{Expression}$$ App Expression Expression
BoolType ::=  $$\mathtt{B}$$ TBool
Arrow ::= $$ \mathrm{Type}\ \to\ \mathrm{Type} $$ Arr Type Type
